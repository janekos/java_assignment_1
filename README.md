<h2>Java assignment</h2>
<h3>Components</h3>
<span>Backend: Spark Framework (2.7.2)</span><br>
<span>Frontend: pure HTML5 (ES6)</span><br>
<span>Database: SQLite (3.23.1)</span><br>
<h3>Requirements</h3>
<span>Java 8</span><br>
<span>Modern web browser</span><br>
<span>SQLite needs to be installed on the usable device and assigned path.</span><br>
<h3>How to run</h3>
<ol>
	<li>Download .zip or clone.</li>
    <li>Open as project in eclipse.</li>
    <li>Right-click on project in eclipse. Select "Maven" > "Update project".</li>
    <li>If needed, configure build path to use JavaSE-1.8 (Execution environment)</li>
    <li>Build</li>
    <li>Run</li>
    <li>Launch contents of "frontend" directory somewhere in localhost (IIS was used during development) and use.</li>
</ol>
